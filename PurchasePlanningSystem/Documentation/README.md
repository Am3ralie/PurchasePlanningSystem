# Система планирования закупок

## Описание
Прототип системы для планирования закупок и формирования заказов поставщикам.

## Основные функции
1. Аутентификация (Admin/Manager)
2. Создание заявок на закупку
3. Формирование заказов поставщикам на основе заявок
4. Просмотр заказов
5. Управление пользователями (для администратора)

## Тестовые данные
- Admin: admin / q1111
- Manager: manager1 / w2222

## Технологии
- ASP.NET Core MVC
- MySQL
- Bootstrap 5
