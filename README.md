# PurchasePlanningSystem

## Система автоматизации планирования закупок и контроля товарных потребностей.  
Разработано для оптимизации процессов пополнения запасов, планирования заказов поставщикам и улучшения управления запасами на предприятии.

## Основной функционал
Анализ потребностей: Расчет потребностей в закупке материалов и товаров на основе текущего уровня запасов и исторических данных.
Планирование заказов: Модуль формирования рекомендованных закупок с возможностью корректировки планов перед отправкой поставщикам.
Управление поставщиками: Хранение информации о поставщиках, условиях поставок, сроках и минимальных партиях.
Оптимизация запасов: Инструменты анализа текущих остатков и прогнозирование будущих потребностей для предотвращения дефицитов и избытков.

## Технологический стек
Язык программирования: C#
База данных: SQL 
Архитектура: Клиент-серверная система

## Назначение
ПО позволяет автоматизировать ручные процессы планирования закупок, минимизировать ошибки при расчёте потребностей и ускорить формирование заказов поставщикам.

## Установка и запуск
1. Откройте решение в Visual Studio.
2. Настройте строку подключения к БД в конфигурационном файле.
3. Запустите проект.
